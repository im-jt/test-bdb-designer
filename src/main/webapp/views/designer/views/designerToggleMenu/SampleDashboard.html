26. Apply Filter open_in_newTry in designer file_downloadDownload Sample

Method : sdk.applyDataSetFilter( "bar32",["Year==2010", "Quarter=='Q2'","Sales>=60"]))

Arguments : Component id (e.g "bar32") ,Conditions written in form of array

Use the function to filter dataset of a component

examples:

sdk.applyDataSetFilter("Bar1", ["Year==2010"]);
sdk.applyDataSetFilter("datagrid3", ["CompanyName=='Google' || CompanyName=='Yahoo'" , "ProjectHrs>5000"]);
sdk.applyDataSetFilter("Bar2", ["Year>=2010", "Month=='Feb'", "Week=='2'"]);
sdk.applyDataSetFilter("Bar3", ["Year=={radioYr.Value}", "Week>=2", "Week<=3"]);
sdk.applyDataSetFilter("Grid2", ["ProjectName=='{filter2.ProjectName}'", "EmployeeCount>={hslider2.Value}"]);

To Sort the dataset and limit number of records, write condition like this:

sdk.applyDataSetFilter("Bar1", ["order by Sales asc", "limit 10"]);

All the conditions written in array, are executed with AND(&&) Operator, If you need to use OR(||) operator: Please write the filter conditions like this:

sdk.applyDataSetFilter("datagrid3", ["CompanyName=='Google' || CompanyName=='Yahoo'" , "ProjectHrs>5000"]);

Available Operators are - "> , < , == , != , >= , <= , || , && "

Every Conditions should be enclosed in DOUBLE QUOTES: "CompanyName=='Google'" is a valid condition

Field name shouldn't be kept in SINGLE QUOTE: "'CompanyName'=='Google'" is invalid condition

Field Name should be valid javascript variable names

Some invalid column names are:

  • Javascript identifiers ( var, boolean, number, function, prototype, new, try, for, Object, this etc.)

Preference of operations on dataset :

  1. Calculated fields
  2. Append additional rows
  3. Automanipulation
  4. Filter
  5. Aggregation

Limitations:

  • If a chart has AutoManipulator enabled, filter is applicable only on GroupBy field and Dynamically-Generated-Fields

    Valid case:

    sdk.applyAutoManipulator("column14", "Year", "Month", "Expense", "#A0D468,#9087EE,#F89406,#EF4836,#6C7A89,#4ECDC4,#F5D76E,#BF55EC");
sdk.applyDataSetFilter( "column14", ["Year>2011", "Jan>100"] );
    Invalid Case: 
    sdk.applyAutoManipulator("column14" , "Year", "Month", "Expense", "#A0D468,#9087EE,#F89406,#EF4836,#6C7A89,#4ECDC4,#F5D76E,#BF55EC");
sdk.applyDataSetFilter( "column14", ["Year>2011", "Month=='Jan'"] );

    "Month=='Jan'" is invalid condition here as the field "Month" will not be available in the resultant dataset after automanipulation operation

  • If a chart has AutoManipulator disabled, filter is applicable on GroupBy fields and Original-Dataset-Fields which are coming from connection

    Valid case:

    sdk.applyAutoManipulator("column14", "Year", "Month", "Expense", "#A0D468,#9087EE,#F89406,#EF4836,#6C7A89,#4ECDC4,#F5D76E,#BF55EC");
sdk.applyDataSetFilter( "column14", ["Year>2011", "Month=='Jan'"] );
    Invalid Case: 
    sdk.applyAutoManipulator("column14", "Year", "Month", "Expense", "#A0D468,#9087EE,#F89406,#EF4836,#6C7A89,#4ECDC4,#F5D76E,#BF55EC");
sdk.applyDataSetFilter( "column14", ["Year>2011", "Jan>100"] );

    "Jan>100" is invalid condition here as the field "Jan" is not coming from the connection

28. Apply Aggregation open_in_newTry in designer file_downloadDownload Sample

Method : sdk.applyAggregation("bar32", "Year", "Revenue,Expenses", "SUM,SUM")

Arguments : Component id (e.g "bar32") , Group by node (e.g "Year") , Fields to be aggregated (e.g. "Sales,Profit") , Operations to be applied (e.g. "COUNT,SUM")

Use the function to apply aggregation on dataset of chart/ grid. These are he available aggregation operations - SUM, AVG, MAX, MIN, COUNT

  • if left blank, only single record will come in result with aggregated values for mentioned Fields 
    sdk.applyAggregation("bar32", "", "Revenue", "SUM");
  • If one group by field is given, result will be grouped according to unique values of the fields(Year in the example) 
    sdk.applyAggregation("bar32", "Year", "Revenue", "SUM");
  • If more then one field is given, result will be grouped by last field(Month field in the example) and then grouped by previous field(Year field in example) 
    sdk.applyAggregation("bar32", "Year,Month", "Revenue", "SUM");
