Tooltip

1. Under component property section, ‘Tooltip ’ property block is available.

2. Tooltip block contains ‘Show Tool Tip ’, ‘Background Color ’, ‘Opacity ’, ‘Border Color ’, ‘Box Width ’, ‘Precision ’ & ‘Highlighter ’.

a) Show Tool Tip: Configures tooltip as Default, Custon or None.

b) Background Color: Configures tooltip background color.

c) Opacity: Configures tooltip background color opacity.

d) Border Color: Configures border color of tool tip.

e) Box Width: Configures width of the tool tip box it contains a select box from where width can assign to tool tip box.

f) Precision: Configures the tool tip display data precision.

g) Highlighter: On enable this property through checkbox tooltip highlighted field background color change.



3. Show Tool Tip: On click this a model box open which contains three radio buttons, its default selection is ‘Default ’

a) None: On selection of None, tooltip doesn't be shown on the selected component.

b) Default: It's by default selected tooltip, displays the default configuration of the selected component.

c) Custom: On selection of Custom, tooltip content can be configured.



4. Custom Tool Tip: On selection of Custom, a editable text box open.

5. Editable text box contains property buttons on top, a select box which contains all the fields of data set, a save and cancel button.

6. Editable text box contains selected fields of data set by default.



7. Custom Tooltip configuration

a) Additional fields can be selected from the select box.

b) Display name then field name inside the bracket, field data replaces brackets.

c) After selection of the content font properties like font color, font size, font family etc can be changed from property buttons.



e) ‘<htmlTag></htmlTag>’ : With the help of this tag, any HTML content can be passed directly into tooltip.

f) Video and image can be displayed in to tool tip through this htmlTab.

g) The field which contains video links that name would be written in the brackets after "scr=" for example src=[Field Name].For displaying single video or image the only link to that particular video or image need to give under "src=" for example src="https://www.bdbizviz.com/resources/images/BDB_logo.png"

h) Font Color property does not support in Internet Explorer.

i) For changing the field data font property, need to select entire bracket otherwise value won't be display in the tooltip.



8. Mobile Tooltip configuration

a) For Timeline, Bar, Heatmap & Pie mobile specific Tooltip is available.

b) This Tooltip works only when component is in maximized state.

c) Default Tooltip can be enable through this script--> sdk.getWidget( 'ComponentID' ).m_mobiletooltipstyle = "default";
