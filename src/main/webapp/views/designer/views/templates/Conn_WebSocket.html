Connecting to a Web Socket

Click the ‘Data source’ icon provided to the right side of the canvas, and view the database connectivity options.

  • Click the ‘Data Source’ icon to display all the available data sources.
  • Click the ‘Add’ button provided next to the ‘Web Service’ option.
  • A new connection will be created.


  • Click the connection to display the connection specific details.
  • Configure the following information:
    • a.Name: Name the connection.
    • b. URL : Paste the url from which websocket data should be pulled.
    • c. Load at Start: Enable this option to get up-to-date data / pull the updated data from the data source before loading into charts.

Guidelines for using a WebSocket

  • A valid Web Socket URL should be given.
  • Fields and Filters are not listed since a Web Socket can send data in any structure.
  • To display the reponse in a chart or table, Parse the response and format the data as per the structure.
    Examples:
    var response = sdk.getWebSocketResponse()
    response.message.dashboard_1.Device; // can be an array of objects
