Connecting to a Merged Data

  • Click the ‘Data Source’ icon to display all the available data sources.
  • Create new connections using any data connector option (In this case, Excel connections are created. E.g. Connection-1, Connection-2)
  • Create a connection under ‘Merged Data’.
  • Configure the new connection created for Merged data by mapping the created Excel connections.
  • Define a condition via the ‘Criteria’ space. (E.g. ‘Equal’ or ‘Not Equal’)
  • Fields for the Merged Data connection will be displayed on the right side of the panel.


Note: Users can ‘Rename’ or ‘Remove’ the fields added for merged connection.
