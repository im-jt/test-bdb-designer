Alert Scorecard

Creating an Alert in Scorecard

1. Drag and drop the Scorecard and assign connection to it.

Fig: Scorecard with data connection

2. Select the field & change 'Column Type' to 'Numeric', to which indicator has to be applied


3. Click on the ‘Indicator’ tab to configure alert settings:


Configure the following Alert properties:

  • Show Alert: Enable this option to view the alerts on scorecards

  • Alert Column: The column on which alerts should be applied

  • Alert by: There are 3 options to view alerts:

    1. Comparison: Used to compare a column against the value given in the ‘Compare Value’ box

    2. Range: Used to set the color based on where the column’s value falls within a given range.

    3. Static Comparison: Used to compare a column against static values given in the ‘Value’ text area.

  • Custom Properties: Configuration for Comparison and Alert options are given here.



  • Show Data: Enable this option to view the data
