Pivot Chart

The Pivot grid also known as Pivot Table is a data summarization tool used in data visualization software. Users can identify and select columns and rows of data in a data base or spreadsheet to obtain a desired report using the pivot grid tool. A pivot grid does not change the actual database or spread sheet, but turns the data in such a way that it can be viewed from the different perspectives.

Best situation to use Pivot Grid:

  • When users want to perform analysis on multi-dimensional data by generating cross- tabular reports.
