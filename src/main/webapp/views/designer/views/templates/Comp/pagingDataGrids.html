Paging Data Grids

Paging Data Grid is used to represent data in tabular form. Paging data grids can be used to filter data using columns, comparison operators and search option provided.

Best situation to use Paging Data Grid:

  • When end-users need to view data in numerical format.
  • When alerts needs to be shown.
  • When end-users needs to filter, compare and search for data within the grid .

Variations

  • Add-up Filters: Filter data by selecting any column(s) present in the data grid
  • Compare operation: Data can be compared by using comparison operators
  • Search: Search for a particular value
