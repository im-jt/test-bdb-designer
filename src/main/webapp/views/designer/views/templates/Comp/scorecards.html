Scorecards

Scorecards are similar to Data grids; where data can be displayed in hierarchical format, and alerts can be set.

Best situation to use Scorecard:

  • When end-users need to view data in numerical format.
  • When alerts needs to be shown.
  • When to display hierarchies in data.

Variations:

  • Compare: A 'Compare Value' must be given by the user. Alerts are then triggered depending on if the data in the data set is greater, equal, or less than the compare value. (Ex. <50% = Red, 50% = Yellow, >50% = Green)
  • Range: Alerts can be configured to trigger based on range value(s). (Ex. 0-40% = Red, 41-70% = Yellow, 70-100% = Green)
